# Encryption
Это техническая часть проекта по физике. тут я покажу на примере кода как работает математическое и квантовое шифрование на деле. код будет на python.

1. Открытая экспонента (e) - это число, используемое в операции возведения в степень при шифровании данных по формуле C=M^e(mod n). Она является частью открытого ключа (e,n). Она напрямую определяет скорость шифрование и безопастность. Если понизить открытую экспоненту (например на 3), то скорость шифрования вырастет, но система станет уязвимой к атакам трансляции (broadcast attack), если одно и то же сообщение отправляется нескольким получателям.
2. Вычисление секретной экспоненты (d): для классического шифрования применяется формула "C=M^e(mod n)", в моём случае это означает, что мне пришлось бы ставить число 12345 в степень 65537, в результате получилось бы огромное число, которое просто напросто забила бы оперативную память. Посему я использовал функцию "pow(M, e, n)", его преимущество щаключается в том, что он производит операцию mod n на каждом шаге вычислений: Степерь переводится в двоичную систему, программа идет по битам степени и если бит равен 0 - число возводится в квадрат, если бит равен 1 - число возводится в квадрат и умножается на основание. После каждого действия результат сокращается по модулю n.
3. Модуль системы (n) - определяет максимальный размер сообщения. От сюда вытекает условие: "m < n", в противном случае, при вычислении остатка по модулю часть данных будет безвозвратно потеряна, и дешифрование вернет неверный результат.
